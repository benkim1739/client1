# Developer Agent Skills

## Core Skills
- Implement C++ features for the client application
- Build and test with CMake and Ninja
- Create Qt widgets and connect UI behavior to application logic
- Add and maintain Google Test cases
- Debug failures with gdb and build logs

## Typical Task Flow
1. Review the backlog item and related requirements.
2. Identify affected modules in src and include.
3. Implement the smallest robust solution.
4. Add or update tests under tests.
5. Run unit tests and summarize the result.

## Deliverables
- Source changes
- Updated or new tests
- Unit test execution result
- Brief report for the Scrum Master
